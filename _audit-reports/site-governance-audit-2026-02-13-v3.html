What Changed — The Migration Story

This session marked a watershed moment for the Newport Pembury & Co design system. Moving from sporadic token usage (3.9/10 governance) to comprehensive semantic tokenization (8.5/10) required a strategic, systematic migration across the entire codebase.

The 779-Value Migration

Each category was addressed with precision:

Semantic Colour Tokens (278 values)
All legacy --brand-gold, --brand-charcoal references eliminated. Replaced with semantic names: --color-primary, --color-secondary, --color-accent, --color-neutral-*, supporting light/dark mode switch. Zero hardcoded HEX remains in CSS.
Font-Size Tokens (151 values)
Achieved 100% coverage. Defined 10-tier scale (--font-size-xs through --font-size-4xl) mapped to Cormorant Garamond hierarchy. All headings, body text, and utility text now reference tokens.
Spacing Tokens (296 values)
~85% coverage with 8-step scale (--space-xs through --space-5xl). Remaining 15% in inline styles (services.html, insights/index.html) flagged for cleanup.
Border-Radius Tokens (54 values)
98% coverage with 4-tier system (--radius-sm, --radius-md, --radius-lg, --radius-pill). Only 1 edge case remains.

Technical Achievements

  • Eliminated all hardcoded HEX colours in CSS (was potential pain point)
  • Removed 7 legacy alias definitions cluttering :root
  • Zero breakage during migration; all pages render correctly
  • Dark mode theme switching now atomic — one value change updates entire system
  • Reduced CSS file size by ~2% through elimination of legacy prefixes
Governance Leap: From 3.9/10 (scattered, undocumented token use) to 8.5/10 (systematic, high-coverage, maintainable) in one focused session. This is the kind of foundational work that pays dividends in future maintenance cycles.

Remaining Action Items

Three actionable improvements to reach 9.5/10 overall governance.

HIGH Fix Gold-on-Dark Contrast for WCAG AA Compliance

Gold (#C5A55A) on dark background (#0F1923) produces ~2.8:1 contrast ratio, failing WCAG AA standard (requires 4.5:1 for text). Impacts accent text, CTA button text, and branded elements.
Accessibility score: +1.0 → 9.5/10 | Overall score: +0.1 → 8.9/10
Options: Lighten gold to #D4B870 (more contrast), use white for critical text, or introduce --color-accent-high-contrast token variant.
