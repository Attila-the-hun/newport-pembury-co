  1. CRITICAL

    • --brand-gold → --color-action-primary (92 refs)
    • --brand-charcoal → --color-text-primary (60 refs)
    • --brand-dark → --color-bg-inverse (25 refs)
    • --brand-ivory → --color-text-inverse (12 refs)
    • --brand-slate → --color-text-secondary (33 refs)
    • --brand-cream → --color-bg-secondary (8 refs)
    • --brand-gold-light → --color-action-hover (15 refs)
  2. CRITICAL
    Replace 127 hardcoded font-size values with var(--font-size-*)
    Font-size coverage is only 15.9%. Scale tokens exist but are unused. Estimated score gain: +1.0 point.
    • Map common sizes: 1rem → --font-size-base, 0.875rem → --font-size-sm, 1.125rem → --font-size-md
    • Create exception tokens for outliers (e.g., 1.75rem, 0.6875rem)
    • Apply consistently across all page types
  3. MAJOR
    Replace 294 hardcoded spacing with var(--space-*)
    Spacing coverage at 23%. Establish 8px grid enforcement. Estimated score gain: +1.0 point.

    • 32px → --space-xl, 48px → --space-2xl
    • Apply to all padding, margin, gap properties
  4. MAJOR
    Replace 44 hardcoded border-radius with var(--radius-*)
    Border-radius coverage at 24.1%. Estimated score gain: +0.5 points.

    • Validate visual consistency after each batch
  6. MINOR
